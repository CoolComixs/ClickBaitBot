# Clickbait Bot
Welcome to the Clickbait Bot made by Mitchell Reiff.
The Bait Bot is a program that:
  - Generates fake headlines
  - Posts the headlines to twitter
  - Made in Java
  - Used [TwitterJ4]
  - [Follow the bot on twitter and use #TheBaitBot!]
